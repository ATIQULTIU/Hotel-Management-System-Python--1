# 🏨 Hotel Management System (Python)

A simple **Hotel Management System** built with **Python, Tkinter, and SQLite**.  
This application allows hotel staff to manage rooms, bookings, checkouts, and customer records through a graphical interface.

---

## 🚀 Features

- 🔐 Admin Login System
- 🏨 View Available Rooms
- 📖 Book a Room
- 💳 Checkout System with Bill Calculation
- 👥 View Client Booking Status
- 🗄 SQLite Database Integration
- 🖥 Simple GUI using Tkinter

---

## 🛠 Technologies Used

- Python
- Tkinter (GUI)
- SQLite3 (Database)
